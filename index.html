Help

How do I import the result into Fritzing?
Download the .fzpz file, then in Fritzing go to Part → Import… and select it, or drag the file onto the "My Parts" bin.
How do pin positions work?
Each view (breadboard/schematic/pcb) has its own background image plus a layer of draggable red dots — one per pin. Drag a dot onto the matching spot on the artwork; its position is baked into the exported SVG automatically.
Can I use my own artwork instead of AI-generated?
Yes — click Upload Image under any view to replace its background with your own PNG/JPG/SVG, then reposition the pin dots on top of it.
Will my API key be remembered next time?
If "Remember this key on this device" is checked, it's saved in this browser's local storage so you won't need to re-enter it. Uncheck it (or clear it manually) to remove it.
Can I resume later / make several parts?
Save Project downloads a full snapshot (including artwork and pin positions) to resume anytime. For multiple parts, click Save to Library after finishing one, then Export All as Bundle when done.
