Help

How do I import the result into Fritzing?
Download the .fzpz file, then in Fritzing go to Part → Import… and select it, or drag the file straight onto the "My Parts" bin.
The SVG shows "MISSING PIN(S)" — what does that mean?
Each connector in the pin table needs a matching element (id="connector0pin", etc) inside the SVG. If the model left one out, click Auto-fix on that view, or edit the SVG code directly in its box.
Can I resume later?
Click Save Project to download a .json snapshot of everything (fields, pins, SVGs, XML). Use Load Project to pick that file back up anytime, on any device.
Can I edit an existing part instead of starting fresh?
Use Import Existing .fzp / .fzpz at the top to load a part you already have — its metadata, pins and views will populate below for editing.
Making several parts at once?
After finishing one, click Save to Library, then start the next with Reset. When you're done, use Export All as Bundle in the Library panel to get every part in one .zip of .fzpz files.
